2 / AlignNet Route

新主线：AlignNet 残差融合，而不是直接大回归

按现有设计文档，这条线的关键不是换 backbone，而是把 hand / scene / object 组织成显式 memory，再交给 `align_hand_net` 做 residual-only 更新。

一句话

Frozen STream3R 提供 camera/depth/world scaffold，HaMeR 提供 hand observation base，scene/object 只作为额外对齐信息，最后统一走 AlignNet residual refinement。

01 / Base observation

HaMeR hand base

  • 先把 crop-camera 输出理顺到 full-image `C_t`。
  • 世界坐标 lift 统一走 `T_{W←C_t}`。
  • 第一阶段只验证“相机接口是否对齐”。
02 / Scene memory

STream3R scene tokens

  • 主干不动，只读取 camera / depth / world scaffold。
  • 先试 full-frame，再收缩到 hand ROI + confidence。
  • Stage-2 当前 winner 是 ROI + confidence 方案。
03 / Object memory

Static + Render 双路 token

  • object 侧拆成 static prior 和 render-conditioned cue。
  • `O_static` 负责 canonical shape prior。
  • `O_render_t` 负责 pose-conditioned visible geometry。
04 / Residual update

align_hand_net

  • 只输出 `Δθ / Δβ / ΔT_{C_t←H}`。
  • 不做 direct regression，避免覆盖基础分支。
  • 后续再接 temporal / infiller，而不是现在就一起上。
