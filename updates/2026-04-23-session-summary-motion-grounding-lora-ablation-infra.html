Key discoveries

FINDING Inference H/W ≠ dataset-native

logs/exp1_4rc_dexycb .npz shows view_X_img is (1, 3, 378, 504). 4RC's load_images(size=512) preserves AR over DexYCB's 480×640 → 378×504 (27×36 patches × 14). Dataset returns 518×518 but that's never what 4RC inference actually feeds.

FIX Stage 2 default --num-epochs broke single-seq overfit

Default 200 caps runs at 200 steps (1 clip × 1 batch per epoch with max-manifest-size=1). Stage 1 defaults 10000. Symptom was 2 jobs 'SUCCEEDED' in 17 min at step 180 instead of 3000.

FIX set -uo pipefail masks torchrun crash as ACP SUCCEEDED

Without -e, a 429-free exit-1 from torchrun lets the trailing `echo 'training complete'` set bash's final exit to 0. ACP records SUCCEEDED for a job that never trained. Always use set -euo pipefail.

FIX sample_step_shape off-by-one on T range

rng.randint(2, t_max + 1) with Python random.Random is inclusive on both ends → yields [2, t_max+1]. With t_max=16 we observed T=17 in history.json. Fixed to rng.randint(2, t_max).

FIX Motion-init loader assumes separate topology

scripts/hoi3r_stage1_overfit.py:66 called model.hand_motion / model.obj_motion unconditionally. Under --motion-topology combined, those attributes don't exist — AttributeError crashed pt-vb73r5xg. Fixed to branch on args.motion_topology and load into combined_motion for the combined path.

FINDING Two-forward scene-loss was wasteful

The first scene-loss impl (09bb7a0) ran a second vanilla arc._forward() just for depth+camera heads. 2× ViT-G wall-time. Refactor f857235 captures intermediate feats at 4RC's hardcoded out_layers=[19,27,33,39] in a single forward; DualDPT consumes them directly. Bit-identity proven on fg path (max_abs_diff=0 on Z/C/T_tok/Q_hand/Q_obj). LoRA grad flows through shared feats.

FINDING Workspace quota is shared across clusters

workspace_aec2_total is summed across computing-cluster-01e + computing-cluster-01e-02 + a third cluster. Cluster selection targets a specific machine pool but can't bypass a 429. A tight poll-and-submit loop with <5s gap wins most races against other teams.
