Analysis

VALIDATED Chamfer + latent: best combination for deformation learning

Two-phase convergence: (1) steps 1–300, latent loss drives feature alignment, CD improves steadily; (2) steps 300–500, latent loss converged, mesh loss continues refining surface quality. No oscillation at any point.

OBSERVATION Mesh loss ceiling at ~0.008

The mesh chamfer loss plateaus at 0.008 (from 0.016). This corresponds to CD ~0.044. Further improvement likely requires: more training steps, multi-frame data, or architectural changes to break the spatial structure limitation of the SLat bottleneck.

NEXT Multi-frame training

Single-frame overfitting reaches CD=0.044. To learn generalizable deformation, train on multiple frames with different hand poses from the same sequence. The canonical SLat is shared; only the image (providing pose info) and GT posed mesh change per frame.
