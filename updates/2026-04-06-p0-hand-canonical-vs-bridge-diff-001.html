Context

Experiment goal

不再通过 encoder/decoder 看症状，而是直接比较输入本身。拿旧 `vae_test_initial` 的 canonical hand OBJ 和 5 个 DexYCB bridge hand，逐项检查 topology、bbox 归一化、最佳 rigid/similarity alignment，以及对齐后残差，回答“bridge hand 和旧 canonical hand 的差异到底在什么地方”。

Conclusion

PASS。当前最稳的输入侧结论是：bridge hand 与旧 canonical hand 的差异至少包含两层，左手 case 里的 topology/handedness 差异，以及即使在 right-hand case 中仍然存在的显著非刚体形状/姿态差异。坏结果不能再归因于“只是整体旋转”。

Key findings

  • 2 个 right-hand bridge 样本与 canonical hand 的 face 数组完全一致；3 个 left-hand bridge 样本则不一致
  • 这说明 left-hand case 里除了几何差异，还额外混入了 left/right template or topology 差异
  • 关键点在 right-hand 子集：即使 topology 完全一致，最佳 rigid alignment 之后 RMS 残差仍约为 `0.156 - 0.176`，均值 `0.166`
  • 因此 bridge hand 和旧 canonical hand 不是单一 rigid transform、也不是简单 similarity transform 的关系
  • 最接近 canonical 的 bridge 样本是 `20200813-subject-02/... frame 24`，但它的最佳对齐 RMS 仍有 `0.1556`
  • overlay 可视化进一步显示：bridge hand 与 canonical hand 在手指展开/弯曲和整体手型分布上仍然存在明显差异
