DexYCB 5seq Hand PCA Canonicalization

hand mean pred/gt vertex ratio dropped from `2060.89x` to `220.41x` and hand mean active voxels dropped by `9.35x`; object ratios stayed unchanged. Strong evidence that missing hand rotation canonicalization before voxelization is a major cause of the previous explosion

Context

Experiment goal

验证 hand 的极端膨胀是否主要来自“只做平移缩放归一化、没有做旋转 canonicalization”。具体做法是：保持 5-seq officialized baseline 的样本、权重和 object 路径不变，只在 hand 输入上增加 `normalize -> PCA align -> normalize`，然后继续走同一条 officialized `vxz` 路径。

Conclusion

PASS。只对 hand 增加 PCA canonicalization 就能把 5-seq officialized 的 hand 膨胀整体压低约一个数量级，同时 object 结果基本不变。这是目前最强的因果证据，说明 hand 朝向 / canonicalization 才是主要敏感因素。

Key findings

  • hand 平均 `pred_to_gt_vertex_ratio` 从 `2060.89x` 降到 `220.41x`
  • hand 平均 `active_voxels` 从 `1,603,372.6` 降到 `171,481.2`
  • 平均 ratio reduction 和 voxel reduction 都约为 `9.35x`
  • object 平均 `pred_to_gt_vertex_ratio` 保持不变，仍为 `27.84x`
  • 5 个 hand 样本的新 ratio 分别为 `212.69x`, `208.32x`, `282.91x`, `233.98x`, `164.17x`
  • 这证明 `vxz` 不是主因，hand 在 voxelization 前缺少旋转 canonicalization 才是主要问题来源
  • 但 hand 仍然没有完全回到旧 `vae_test_initial` 的 `178k` canonical hand 量级，所以 PCA 对齐解决的是大头，不是最终完整解
