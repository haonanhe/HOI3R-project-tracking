Task A — Scene Distillation Loss

New module stream3r/loss/scene_losses.py implements a distillation loss that compares predicted scene point clouds against STream3R-generated pseudo-GT world points. Wired into dcf_full_sequence_train.py via the --loss-weight-scene flag (default 0.0; set to 1.0 for this smoke).

Code Changes

  • stream3r/loss/scene_losses.py — new file, implements SceneDistillationLoss
  • dcf_full_sequence_train.py — added --loss-weight-scene arg, imports and calls scene loss in training loop

Task C — Generalization Range

Added --train-range and --eval-range arguments to dcf_generalization_test.py, supporting contiguous ranges like "0-4" in addition to the existing comma-separated indices.

Code Changes
