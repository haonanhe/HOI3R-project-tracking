Task A — Real Scene Loss

What Changed

Replaced pseudo-GT distillation with real DexYCB ground-truth supervision via depth_loss + camera_loss, following the canonical STream3R loss pattern at stream3r/loss/losses.py:1777. The scene loss now backpropagates real depth and camera errors from the DexYCB dataset rather than distilling from a frozen teacher.

Task C — Generalization Range

What Changed

Added --train-range / --eval-range CLI flags with end keyword support, mutually exclusive with the older --train-frames / --test-frames flags. Overlap between train and eval ranges is validated at startup and rejected with an error. Resolved splits: train [0, 1, 2, 3, 4], eval [5, 6, 7, 8, 9].
